Sistema de Cadastro (Python + MySQL)
Sistema de cadastro de usuários desenvolvido em Python com integração a banco de dados MySQL e interface no terminal.
O projeto simula um sistema CRUD completo com foco em organização, validação de dados e boas práticas de desenvolvimento.

🚀 Demonstração
- Menu interativo no terminal
- Cadastro de usuários
- Listagem de registros
- Atualização de dados
- Exclusão de usuários
- Validação de email (somente Gmail)
- Senhas criptografadas com bcrypt
- Sistema de login

🧠 Objetivo do projeto
Este projeto foi criado para praticar:
- Estruturação de sistemas CRUD
- Integração Python + MySQL
- Validação de dados
- Segurança básica (hash de senhas)
- Organização de código em funções
- Experiência com bibliotecas externas

🛠️ Tecnologias utilizadas
- Python 3
- MySQL
- rich (interface no terminal)
- bcrypt (criptografia de senha)
- re (validação de email)



Projeto desenvolvido para fins de estudo em Python e banco de dados.
